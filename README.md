# DeepFake-Detector-EXT

## Configurare API
Codul nu conține o cheie API predefinită, deoarece folosesc versiunea gratuită oferită de **Google AI Studio**.

Pentru a testa demo-ul:
1. Accesați [Google AI Studio - API Keys](https://aistudio.google.com/api-keys).
2. Creați un cont.
3. Utilizarea este gratuită; nu se aplică taxe suplimentare.

## Informații despre Proiect
Acest proiect reprezintă un **demo**.

În prezent, pentru detecția de deepfake-uri, am utilizat `gemini-latest`. Totuși, consider că o implementare bazată pe **Grok AI** (de la x.ai) ar fi mult mai eficientă, având în vedere capacitățile sale de tip "Thinking" (raționament), spre deosebire de modelele standard de generare de text.

### Potențial și Upgrade
Utilizarea unui API plătit de la Grok ar putea transforma acest proiect într-o soluție mult mai robustă. Deși interfața necesită actualizări suplimentare, acest cod servește drept bază demonstrativă.

* **Costuri estimate Grok API:** Aproximativ 2 USD per 2 milioane de token-uri.
* **Scalabilitate:** Acest volum este suficient pentru a susține un număr mare de cereri zilnice.
* **Resurse:** Consultați [site-ul oficial x.ai](https://x.ai) pentru detalii actualizate despre prețuri.
